FAQs

  • How do I add a new item? Go to the Inventory page and click "+ Add Item."
  • How do I place an order? Navigate to the Orders page, select a supplier and product, then submit.

Troubleshooting

Images not loading: Ensure the image files are in the correct directory (`images/`). Check the console for errors.

Firebase data not loading: Verify your Firebase configuration and network connection.

Glossary

Inventory: List of items in stock.

Supplier: A vendor providing products.

Expiration Date: The date after which an item should not be used.
